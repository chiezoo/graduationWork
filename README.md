##　テーマ変更の背景
元々は架空の病院サイトの構築を検討していましたが、以下の理由を持ってテーマ変更と致しました。
架空の病院サイトで使い勝手の確認をしてもらいにくい
データの連携が少なく、ER図での表現が希薄になる
病院スタッフ側になり、手動で番号を進めたり、患者側になったり、テストしくい
